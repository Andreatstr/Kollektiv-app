# Shippable produkt

Vi har brukt jlink og jpackage til å generere en java-runtime og en .exe fil for å installere appen vær lokalt på pcen. Denne metoden er testet for Windows



## Viktige installasjoner for at det skal funke
- WiX Toolset versjon 3.14
- JavaFX SDK versjon 23.0.1
- JavaFX jmods versjon 23.0.1
